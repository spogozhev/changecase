# Конвертер регистров
Учебный проект.

Конвертер регистров позволяет менять в тексте:
- ВЕРХНИЙ РЕГИСТР на нижний регистр,
- нижний регистр на ВЕРХНИЙ РЕГИСТР.

Поддерживается замена только для английских букв из таблицы ASCII.

## Командная строка
```
  changecase -key source_filename destination_filename
```
Поддерживаемые ключи:
- `-U` – преобразование текста в ВЕРХНИЙ РЕГИСТР,
- `-L` – преобразование текста в нижний регистр,
- `-G` – переключение регистров (из ВЕРХНЕГО в нижний и наоборот),
- `-T` – каждое слово начинается с заглавной буквы,
- `-R` – регистр каждого символа определяется случайным образом.
